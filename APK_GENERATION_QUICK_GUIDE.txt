QUICK STEPS (Copy & Paste Commands)
================================================================================

STEP 1: Create GitHub Repository
  - Go to https://github.com
  - Click "+" → "New repository"
  - Name: easy-life-keyboard
  - Click "Create repository"
  - Copy the URL shown (you'll need it next)

STEP 2: Upload Project to GitHub
  
  Open Terminal/Command Prompt and run:
  
  cd path/to/easy-life-keyboard-native
  git init
  git add .
  git commit -m "Initial commit: Easy Life Keyboard v1.0"
  git remote add origin https://github.com/YOUR_USERNAME/easy-life-keyboard.git
  git branch -M main
  git push -u origin main

STEP 3: Enable GitHub Actions
  
  - Go to your GitHub repository
  - Click "Settings" tab
  - Click "Actions" → "General"
  - Select "Allow all actions and reusable workflows"
  - Click "Save"

STEP 4: Trigger Build
  
  - Go to "Actions" tab
  - Click "Build APK" workflow
  - Click "Run workflow"
  - Select "main" branch
  - Click "Run workflow"

STEP 5: Wait for Build (5-10 minutes)
  
  - Watch the workflow status
  - Yellow = Building
  - Green = Success
  - Red = Failed

STEP 6: Download APK
  
  - Click the completed workflow
  - Scroll to "Artifacts" section
  - Download "easy-life-keyboard-debug.zip"
  - Extract the ZIP
  - You'll find "app-debug.apk"

STEP 7: Install on Android Device
  
  - Connect phone via USB
  - Copy APK to phone
  - Tap APK to install
  - Grant permissions
  - Done!

STEP 8: Set as Default Keyboard
  
  - Open Easy Life Keyboard app
  - Tap "Set as Default Keyboard"
  - Go to Settings → Languages & Input → Keyboard
  - Select "Easy Life Keyboard"

================================================================================
TROUBLESHOOTING
================================================================================

Build Failed?
  → Check GitHub Actions logs for errors
  → Ensure all files are committed
  → Retry build

Can't Find APK?
  → Build must complete successfully (green checkmark)
  → Scroll down to "Artifacts" section
  → Look for "easy-life-keyboard-debug"

APK Won't Install?
  → Uninstall old version first
  → Check device has enough storage
  → Enable "Unknown Sources" in Settings

App Crashes?
  → Check Android version (10+)
  → Reinstall APK
  → Check storage space
